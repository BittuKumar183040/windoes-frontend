# Windoes

A web-based portfolio project that recreates the Windows 11 desktop experience in the browser. This project showcases various applications and features, including:

- **Login**: Authintication using username and password or using pin.
- **Registration**: New user onboarding process of creating account to the desktop.
- **Notepad**: Simple text editor for quick notes.
- **Paint**: Basic drawing and painting tool.
- **3D Viewer**: Preview and interact with 3D models.
- **Future Enhancements**: 
  - More applications and features planned.
  - User level customization 

## Features

- Windows 11-inspired UI/UX
- Responsive and interactive desktop environment
- Modular app integration

## Getting Started

1. Clone the repository:
  ```bash
  git clone https://github.com/yourusername/windoes.git
  ```
2. Install dependencies:
  ```bash
  npm install
  ```
3. Start the development server:
  ```bash
  npm run dev
  ```
